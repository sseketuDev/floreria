Carpeta para fotos de clientes.
Puedes guardar imágenes aquí y luego usar `index.html` para referenciarlas si prefieres cargar imágenes desde el sistema de archivos en vez de subirlas en tiempo real.
Nombres sugeridos (opcional):
- cliente1.jpg
- cliente2.jpg
- cliente3.jpg

Nota: El carrusel en la página permite subir imágenes desde el navegador (no sube al servidor). Para usar imágenes locales en el carrusel sin subir, coloca los archivos en esta carpeta y referencia su ruta en el HTML si lo deseas.
